<Section icon={Target} title="理解复古操作系统界面">

在这个复古心理恐怖游戏的数字化世界中探索，需要敏锐的眼光和分析性思维。**Desktop Explorer 玩法**的核心围绕着一个运行 90 年代复古操作系统的模拟 CRT 显示器展开。为了推进谜题，你必须像对待真实的电脑一样对待这个虚拟桌面，在层层嵌套的目录中搜寻，运行系统工具，并分析失踪亲人留下的碎片化用户文件。

与现代系统不同，这个复古界面依赖于手动文件管理。你无法简单地搜索文件；你必须定位到确切的目录路径，修复损坏的扇区，并利用眼前隐藏的线索来解密锁定的归档文件。

<Callout type="tip" title="系统导航技巧">
始终检查文件扩展名。某些文件看起来像系统配置文件，但实际上是重命名后的文本文件，其中包含关键密码或背景设定细节。
</Callout>

为了帮助你更高效地探索系统，请熟悉桌面上可用的主要应用程序：

| 应用程序 | 功能 | 优先级 |
| :--- | :--- | :--- |
| 文件资源管理器 (File Explorer) | 浏览目录，打开文本文件，并寻找隐藏的系统文件夹 | 高 |
| 聊天客户端 (Chat Client) | 阅读存档的对话记录，拼凑出角色之间的关系 | 高 |
| 回收站 (Recycle Bin) | 恢复已删除的文档、系统日志和损坏的媒体文件 | 中 |
| 媒体播放器 (Media Player) | 播放包含环境线索的复古音频剪辑和视频文件 | 中 |
| 解密工具 (Decryption Tool) | 输入十六进制密钥和密码以解锁安全目录 | 高 |

理解这些程序之间如何交互是游戏核心循环的基础。例如，在聊天记录中找到的线索可能会提供解密工具所需的密码，随后在媒体播放器中解锁一个损坏的视频文件。

</Section>

<Section icon={Map} title="Desktop Explorer 玩法：逐步通关攻略">

要成功完成主线调查，你必须遵循一条结构化的数字取证路径。本通关攻略涵盖了从初始启动顺序到数字化谜题核心的关键推进步骤。

<Callout type="warning" title="容易错过的线索">
在解密某些系统文件之前将其删除可能会导致你永久无法解锁特定结局。请务必将可疑文件备份到虚拟软盘驱动器中。
</Callout>

按照以下步骤进行游戏前期和中期的调查：

<Steps>
  <Step num="1" title="启动系统并规划目录">
    打开虚拟电脑电源并等待桌面加载。打开文件资源管理器，系统地打开 C: 根目录下的每个文件夹。对任何需要管理员权限的锁定文件夹或文件进行物理记录。
  </Step>
  <Step num="2" title="分析聊天存档">
    打开聊天客户端，阅读用户与其叔叔之间的日志。寻找突出显示的日期、地点和不寻常的词句。在 1998 年 6 月的一段聊天对话中找到的叔叔出生年份，将作为你的第一个主要密码。
  </Step>
  <Step num="3" title="还原回收站">
    导航至回收站并找到名为 `sys_recovery.log` 的文件。将此文件还原到桌面。使用文本编辑器打开它，以显示已删除目录及其原始扇区位置的列表。
  </Step>
  <Step num="4" title="执行解密程序">
    打开桌面上的解密工具。输入从聊天存档中获得的密码以解锁安全系统文件夹。此操作将授予你对隐藏 D: 盘的管理员访问权限。
  </Step>
</Steps>

一旦你解锁了管理员权限，就会出现几个谜题挑战。使用以下密码解决方案来旁路这些安全阻碍：

| 密码位置 | 线索来源 | 解决方案代码 | 解锁内容 |
| :--- | :--- | :--- | :--- |
| 锁定文件夹 1 | 聊天日志日期 | 1958 | 叔叔的个人日记文件 |
| 加密的聊天日志 | 维吉尼亚密码 | RETRO95 | 最终对话存档 |
| 系统设置 | 十六进制代码 | 4F8E | 损坏的注册表文件 |
| 压缩归档包 | 图像文件元数据 | BLUE_SCREEN | 隐藏的音频录音 |

</Section>

<Section icon={Sparkles} title="解密隐藏文件与秘密">

游戏最深层的秘密并不在主线推进路径上。相反，它们被深埋在损坏的扇区和隐藏的系统日志中。发现这些可选文件可以丰富背景设定，并解锁替代叙事路径。

<Callout type="info" title="系统故障">
如果你的虚拟桌面开始闪烁或显示画面撕裂，请不要重启。这些故障是刻意设计的，通常会在桌面屏幕上显现出隐藏的文件夹路径。
</Callout>

要完整地揭开谜团，你必须在虚拟操作系统中寻找特定类型的隐藏数据：

<CardGrid cols={3}>
  <Card title="已删除的文档">
    - **在 Temp 文件夹中寻找**
    - 包含开发者的彩蛋
    - 揭示背景故事
  </Card>
  <Card title="加密的目录">
    - **需要十六进制解密**
    - 访问开发者笔记
    - 解锁秘密音频日志
  </Card>
  <Card title="系统日志">
    - **隐藏在 system32 中**
    - 解释系统崩溃原因
    - 追踪用户活动历史
  </Card>
</CardGrid>

要定位这些文件，你必须使用特定的系统路径。以下是寻找游戏中隐藏最深的秘密文件的指南：

| 秘密文件名 | 文件路径 | 获取方法 | 剧情意义 |
| :--- | :--- | :--- | :--- |
| `project_alpha.txt` | C:/Windows/Temp/log.txt | 运行系统恢复工具 | 揭示操作系统的真实本质 |
| `voice_memo.wav` | D:/Media/hidden_wav | 将文件扩展名更改为 .wav | 解释用户的失踪原因 |
| `family_photo.bmp` | 回收站/img_09 | 还原并用画图工具打开 | 展示关于物理房间的线索 |
| `system_error.log` | C:/Windows/System32/err | 运行诊断程序 | 揭示一个邪恶的 AI 存在 |

</Section>

<Section icon={Shield} title="技术设置与性能优化">

由于这款游戏模拟了复古环境，因此视觉和音频设置对沉浸感和游戏表现都有着巨大的影响。调整这些选项可以帮助你更轻松地发现隐藏的线索，并防止在长时间调查中产生眼部疲劳。

<Callout type="warning" title="视觉舒适度">
CRT 显示器滤镜模拟了历史上的屏幕闪烁。如果你感到眼睛疲劳或头痛，请立即在图形菜单中禁用屏幕闪烁设置。
</Callout>

优化你的模拟环境可确保更流畅的游戏体验。请参考以下推荐的技术设置：

| 设置类别 | 推荐选项 | 游戏体验益处 |
| :--- | :--- | :--- |
| CRT 滤镜 | 开启（低闪烁） | 增强复古氛围并使隐藏文本更易读 |
| 鼠标灵敏度 | 低 | 在选择微小像素元素时允许精确点击 |
| 音频音量 | 高（戴耳机） | 对于听到微弱的背景音频线索和杂音至关重要 |
