<Section icon={Target} title="核心游戏机制与谜题类型">

游戏的核心循环围绕着目录探索和数据解密展开。玩家从简单的目录结构开始，必须定位隐藏文件、阅读文本内容并分析系统日志，以找到打开高级文件夹所需的密码。

谜题旨在测试你的观察能力和逻辑思维。有些挑战需要你修改文件扩展名，而另一些则要求你在模拟的系统注册表中搜索缺失的配置值。

| 谜题类别 | 主要目标 | 难度等级 |
| :--- | :--- | :--- |
| **目录导航** | 在子文件夹中定位隐藏文件 | **简单** |
| **密码恢复** | 利用线索解密文本文件 | **中等** |
| **注册表操作** | 编辑配置键以解锁路径 | **困难** |
| **音频分析** | 解码音频文件中的隐藏信息 | **困难** |

</Callout>
