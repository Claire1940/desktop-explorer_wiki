<Section icon={Swords} title="攻略与谜题解答">

Desktop Explorer 试玩版的核心玩法围绕着绕过安全锁和解决数字逻辑谜题展开。电脑的前任所有者留下了几个加密档案来保护敏感的个人记录。为了将试玩版推向高潮，你必须按特定顺序解决这些谜题。

下表列出了试玩版中遇到的主要谜题、线索位置以及解决方法。

| 谜题名称 | 线索位置 | 解决方法 | 奖励 |
| :--- | :--- | :--- | :--- |
| 用户账户锁定 | 回收站中的便签 | 输入恢复日期 | 访问主目录 |
| 解密工具 | 损坏的图像元数据 | 提取五位十六进制代码 | 解锁聊天记录 |
| 音频文件噪音 | 系统诊断日志 | 调整频率滑块 | 揭示语音录音 |
| 隐藏文件夹密钥 | 聊天室归档 | 组合名字的首字母 | 解锁最后的日记 |

要解决解密工具谜题，你必须仔细重建系统文件夹中发现的损坏图像文件。该文件包含一串隐藏的十六进制代码，对应于网络密码。如果你连续三次输入错误代码，系统将触发安全警报，导致桌面界面扭曲并改变背景音频。

<Callout type="warning" title="安全触发器">
不要盲目猜测密码。每次失败的尝试都会增加系统损坏度，这会导致 CRT 屏幕剧烈闪烁，并可能锁定某些可选的背景故事文件。
</Callout>

</Section>

<Section icon={Shield} title="结局指南与叙事线索">

Desktop Explorer 试玩版根据你对硬盘的调查彻底程度以及你选择删除或恢复哪些文件，设有多个叙事结局。你的行为会留下数字足迹，决定虚拟机的最终状态。

了解每个结局的要求是体验试玩版所有内容的关键。

| 结局标题 | 解锁条件 | 最终结果 |
| :--- | :--- | :--- |
| 系统关机 | 仅完成主线进度谜题 | 电脑正常关机 |
| 失去连接 | 未能解密最后的网络日志 | 出现系统错误消息 |
| 数字实体 | 恢复所有已删除文件和在线日志 | 屏幕闪烁，一个实体开始说话 |

要达成“数字实体”结局，你必须确保硬盘上没有残留的损坏数据。这需要恢复回收站中的每一项，并成功运行系统修复工具。一旦系统完全恢复健康，隐藏在代码中的实体就会显现，更改你的桌面壁纸并传达一条令人难忘的最终信息。

<Callout type="info" title="进度连续性">
你的存档文件会记录你已达成的结局。解锁全部三个结局会触发一个特殊的演职员表后画面，暗示游戏的正式版内容。
</Callout>
