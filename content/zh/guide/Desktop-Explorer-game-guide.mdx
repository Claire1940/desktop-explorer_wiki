安装 Desktop Explorer 后，你将面临一系列需要与桌面文件交互的文本冒险谜题。第一个主要障碍是“古代遗迹（Ancient Ruins）”部分。这里的成功取决于你将“Inheritance”文件夹中发现的线索与游戏窗口中的提示联系起来的能力。

<CardGrid cols={2}>
  <Card title="Inanna 密码">
    在“Long Scroll”文件中找到。文中提到“Inanna 照亮了我的道路”。使用此密码解锁第一个受保护的文件夹。
  </Card>
  <Card title="大象谜题">
    谜题“我用我的软管浇花……我用我的鼻子”指的是**大象**。该提示的具体答案是 **trunk**（象鼻）。
  </Card>
</CardGrid>

解决象鼻谜题后，你会遇到“假墙（Fake Wall）”。许多玩家被困在这里寻找物理按钮。然而，游戏要求你“读懂字里行间的意思”。

<Callout type="warning" title="重要提示">

<Section icon={Target} title="高级文件操作技巧">

**Desktop Explorer 游戏指南**的核心涉及直接在桌面上操作游戏文件。这在“大门”谜题中表现得最为明显，常规钥匙无法奏效。你必须利用“一张图胜过千言万语”的线索来绕过锁。

<Steps>
  <Step num="1" title="定位钥匙文件">
    在下载或谜题文件夹中找到名为 `key.stp` 或类似名称的文件。
  </Step>
  <Step num="2" title="重命名扩展名">
    右键单击该文件并将其重命名为 `image.png` 或 `image.jpg`。这会“欺骗”操作系统显示隐藏的视觉数据。
  </Step>
  <Step num="3" title="提取密码">
    打开新创建的图像。文本“Every lock obeys portonus（每一把锁都听从 portonus）”将会出现。使用 **portonus** 来解锁中央大门。
  </Step>
  <Step num="4" title="检查权限">
    如果系统拒绝重命名，请确保你以正确的用户身份登录，或在属性中检查文件是否为“只读”。
  </Step>
</Steps>

<Callout type="info" title="系统机制">
在该操作系统中重命名文件会创建一个新副本。如果你犯了错误，可以随时删除损坏的副本，并使用原始源文件重试。
</Callout>

</Section>

随着你深入探索，游戏会转向心理恐怖风格。“压倒性的黑暗（Overwhelming Dark）”部分引入了查看隐藏文件的能力。对于想要了解完整叙事的玩家来说，这是 **Desktop Explorer 游戏指南**中的关键一步。

“Fleeting Face”是一个跳杀机制，同时也充当谜题线索。当面孔出现时，它通常会留下一个新文件或修改现有文件。寻找遭遇后出现的 `I know you saw me.stp` 文件；其源代码包含下一组坐标。

<Callout type="warning" title="警告">
查看隐藏文件可能会触发游戏内的系统不稳定。如果“Bizarro”助手开始表现异常，这意味着你正接近一个重大的背景真相。
</Callout>

| 秘密物品 | 位置 | 意义 |
| :--- | :--- | :--- |
| **平石 1-5** | 各个隐藏文件夹 | 拼凑出最终谜题的线索 |
| **卡桑德拉文件** | 谜题 7 目录 | 警告不要继续前进 |
| **Project X** | 根目录 | 已锁定；需要游戏后期的凭据 |

</Section>

<Section icon={Trophy} title="最终谜题：Rubicon 与蛇">

演示版的巅峰涉及“无法逃脱（No Escape）”场景。在折返回入口后，你会发现景观发生了变化。你必须使用之前收集的“平石”在墙上雕刻符号。

<CardGrid cols={3}>
  <Card title="Rubicon 解法">
    谜题 7 背景中的钟乳石形成了字母 **Rubicon**。这是“祭坛（Altar）”部分的密码。
  </Card>
  <Card title="堆叠塔">
    从下往上阅读石头。塔上拼写出“Ever Eating”。这指的是**蛇**（衔尾蛇）。
  </Card>
  <Card title="内存限制">
    如果你收到“内存不足（Out of Memory）”错误，请关闭“Web Surfer”和“Storyteller”窗口以继续。
  </Card>
</CardGrid>

在最后的石柱中输入 **snake** 将触发演示的结束。系统将终止应用程序，声称你“不应该在这里”。这种元结局暗示玩家已经绕过了预期的“演示范围”。

<Callout type="success" title="成就达成">
解决“蛇”谜题被认为是 Desktop Explorer 演示版的“真结局”。它证实了你对游戏逻辑的掌握。
</Callout>

| 最终密码 | 背景 | 结果 |
| :--- | :--- | :--- |
| **Rubicon** | 钟乳石阵 | 解锁礼堂 (Auditorium) |
| **snake** | Ever Eating 之塔 | 结束演示 / 秘密结局 |
| **halibut** | 用户登录 | 默认访问权限 |
