<Section icon={Target} title="主要调查：分步流程">

遵循结构化的 **Desktop Explorer 流程攻略** 至关重要，以确保你不会错过触发特定结局的易遗漏线索。谜团围绕着一个失踪的人和他们留下的数字幽灵展开。你的目标是拼凑出导致这台电脑被遗弃的事件。

<Steps>
  <Step num="1" title="初始启动与探索">
    首先打开桌面上所有可用的文件夹。如果有名为“ReadMe.txt”的文件，请阅读它，因为它通常包含第一层世界观构建。密切关注文件的日期，以建立事件的时间线。
  </Step>
  <Step num="2" title="第一个密码障碍">
    你最终会遇到一个锁定的文件夹或受密码保护的应用程序。在“便签”或浏览器历史记录中搜索聊天记录中提到的常用名称、出生日期或爱好。
  </Step>
  <Step num="3" title="恢复已删除的数据">
    检查回收站。某些文件可能已损坏。你需要在系统内找到“磁盘碎片整理程序”或类似的实用工具，将这些文件恢复到可读状态。
  </Step>
  <Step num="4" title="分析数字证据">
    一旦进入隐藏目录，寻找音频文件或低分辨率图像。这些通常包含关于电脑前主人去向的最直接线索。
  </Step>
  <Step num="5" title="最终对决">
    当你进入等同于“System32”的目录或隐藏的根目录时，游戏即告结束。如果你追求真结局，请确保已收集所有可选文件。
  </Step>
</Steps>

| 章节 | 关键目标 | 难度 |
| :--- | :--- | :--- |
| **启动** | 访问主要用户配置文件 | 简单 |
| **深网** | 在模拟浏览器中导航 | 中等 |
| **加密秘密** | 破解 8 位主代码 | 困难 |
| **结局** | 触发最终系统事件 | 中等 |

<Callout type="warning" title="警告">
有些文件是“陷阱”，可能导致系统崩溃或“坏结局”。在打开可疑的可执行文件之前，请务必保存进度（如果游戏允许，请创建“系统还原点”）。
</Callout>

</Section>

<Section icon={Swords} title="谜题解法：密码与隐藏文件">

**Desktop Explorer 流程攻略** 的核心在于解决利用模拟操作系统功能的逻辑谜题。这些不是传统的物品栏谜题，而是数字侦探工作。

**常见谜题类型：**
- **基于文本的谜语**：出现在电子邮件草稿或聊天记录中。
- **视觉线索**：隐藏在桌面壁纸或图像文件的背景中。
- **音频隐写术**：在损坏的音频剪辑中寻找特定的频率或模式。

| 谜题类型 | 常用解决方法 | 成功率 |
| :--- | :--- | :--- |
| **文件夹锁定** | 日志中的生日/周年纪念日 | ★★★★☆ |
| **应用身份验证** | “笔记”中的安全问题提示 | ★★★★★ |
| **损坏的文件** | 运行内部“修复工具” | ★★★☆☆ |
| **隐藏目录** | 使用“显示隐藏文件”开关 | ★★★★☆ |

<Callout type="info" title="系统洞察">
游戏会根据你打开的独特文件数量来追踪你的“调查等级”。某些“彩蛋”和高级密码只有在你探索了 80% 的可用目录后才会激活。
</Callout>

调查任务要求你有条不紊。如果密码提到了“最喜欢的地方”，请在“图片”文件夹中查找可能揭示地点名称或地标的数字照片。

</Section>

<Section icon={Trophy} title="结局指南：要求与结果">

你的调查有多种结束方式。**Desktop Explorer 流程攻略** 的这一部分详细说明了如何达成每种结果。你的选择——特别是你选择删除或报告哪些文件——决定了数字世界的最终状态。

| 结局名称 | 要求 | 叙事结果 |
| :--- | :--- | :--- |
| **标准结局** | 完成主要文件链 | 基本谜团被解开 |
| **真结局** | 100% 文件发现 + 秘密日志 | 失踪案的完整背景 |
| **损坏结局** | 删除关键系统文件 | 电脑变得无法使用 |
| **观察者** | 观看所有视频日志而不采取行动 | 对这场悲剧的独特视角 |

要达成**真结局**，你必须确保不仅找到了文件，还在调查板上“链接”了它们（如果适用）。这需要深入研究“叔叔”的背景和机器本身的历史。

<Callout type="info" title="结局提示">
“损坏结局”通常是最容易误打误撞得到的。除非你专门在寻找那个结局，否则请避免使用“Format C:”命令，因为它会抹除你在游戏世界内的存档进度。
</Callout>

</Section>

<Section icon={Shield} title="系统要求与技术设置">

Desktop Explorer 旨在现代系统上运行，同时完美模拟 90 年代的环境。为了获得最佳氛围，建议进行某些设置。

| 类别 | 最低要求 | 推荐设置 |
| :--- | :--- | :--- |
| **操作系统** | Windows 10/11 | Windows 10/11 |
| **处理器** | 双核 2.0 GHz | 四核 3.0 GHz |
| **内存** | 4 GB RAM | 8 GB RAM |
| **显卡** | 集成显卡 | 独立显卡（用于着色器） |
| **存储** | 2 GB 可用空间 | SSD 以获得更快的加载速度 |

**沉浸式推荐设置：**
- **CRT 滤镜**：设置为“高”或“真实”以获得最佳复古感。
- **音频延迟**：确保你的扬声器设置为立体声，以便听到方向性音频线索。
- **分辨率**：在 4:3 纵横比的窗口中运行可以增强怀旧效果。

<Callout type="warning" title="技术说明">
如果你遇到“画面撕裂”，这可能是故意的“故障”效果。如果你有光敏性担忧，请在选项菜单中切换“叙事故障”开关。
</Callout>

</Section>
