<Section icon={Map} title="新手入门：操作复古界面">

Desktop Explorer 的核心体验围绕着一个模拟的 1990 年代电脑桌面展开。要推动进度，你必须将该界面视为一个物理空间。在调查中，每个图标、窗口和系统消息都具有特定的作用。与现代操作系统不同，这个环境鼓励玩家进行缓慢、有条不紊的点击，并仔细检查文件的元数据。

| 界面元素 | 主要功能 | 调查价值 |
| :--- | :--- | :--- |
| **我的文档** | 存储文本文件和图片 | 包含主要的故事线索和背景设定 |
| **回收站** | 存放“被删除”的系统文件 | 经常隐藏着加密文件夹的密码 |
| **网上邻居** | 模拟拨号上网连接 | 用于触发特定的故事事件 |
| **控制面板** | 系统设置和显示 | 显现隐藏的文件扩展名所必需的工具 |
| **开始菜单** | 程序访问与关机 | 触发游戏的结局序列 |

桌面环境不仅仅是一个 UI，它本身就是一个叙事谜题。你会发现某些应用程序需要特定的系统日期或“用户权限”才能打开。这些权限通常可以通过阅读前任主人“叔叔”的聊天记录来找到，他的数字足迹定义了整个游戏的谜团。

<Callout type="warning" title="系统警告">
不要忽略“系统错误”弹窗。在 Desktop Explorer 中，这些很少是真正的 bug。大多数错误都包含十六进制代码，这些代码可用作游戏后期发现的加密 .zip 文件的密码。
</Callout>

</Section>

<Section icon={Target} title="Desktop Explorer 全流程通关攻略：步骤详解">

推进故事需要遵循特定的发现顺序。游戏分为逻辑上的“会话”，电脑的状态会根据你的操作而改变。请按照以下步骤操作，以确保你不会遗漏解锁真结局（True Ending）所需的关键证据。

<Steps>
  <Step num="1" title="首次启动与文件审计">
    首先打开“我的文档”中的每一个文件夹。先阅读“ReadMe.txt”文件。它包含一个与“Private”（私密）文件夹密码相关的的时间戳。在桌面背景中寻找任何扭曲的像素，这些地方通常隐藏着数字字符串。
  </Step>
  <Step num="2" title="密码挑战">
    访问“邮件客户端”应用程序。你会找到一系列未发送的草稿。按时间顺序阅读这些草稿的主题行，即可获得 BIOS 密码的提示。输入此密码后，你就可以查看“System32”中的隐藏文件。
  </Step>
  <Step num="3" title="恢复已删除的证据">
    打开回收站并尝试“全部还原”。某些文件会因磁盘错误而还原失败。此时请导航至“磁盘碎片整理程序”工具。运行这个小游戏会揭示“Missing_Fragment.dat”文件的位置，这对于游戏中的剧情反转至关重要。
  </Step>
  <Step num="4" title="聊天记录调查">
    启动仿制的“即时通讯工具”。你需要通过选择正确的提示来模拟对话。目标是从自动机器人的回复中套出“叔叔”的生日。该日期（MMDDYYYY格式）可以解锁“Legacy_Archive”文件夹。
  </Step>
  <Step num="5" title="最终系统覆盖">
    收集完所有故事文件后，会出现一个新图标：“Shutdown.exe”。在点击它之前，请确保你已将“Evidence.log”移动到“根”（Root）目录。此操作是触发游戏最完美结局的触发器。
  </Step>
</Steps>

<Callout type="tip" title="专业提示">
如果你卡关了，可以尝试在控制面板中将“系统配色方案”更改为“高对比度”。这通常会显现隐藏在某些文档空白处的隐形文字。
</Callout>

</Section>

<Section icon={Shield} title="谜题解答与密码目录">

Desktop Explorer 中的谜题旨在测试你对细节的关注。大多数解法都是通过交叉比对两种不同类型文件（例如照片和文本文件）中的信息来找到的。

| 谜题名称 | 位置 | 解谜逻辑 | 密码/代码 |
| :--- | :--- | :--- | :--- |
| **叔叔的保险库** | 桌面归档 | 计算“Work”文件夹中的图标数量 | **8421** |
| **加密 ZIP** | 我的文档 | “Music”文件夹中每个文件的首字母 | **BEETHOVEN** |
| **绕过 BIOS** | 启动画面 | 依次按下 F8、F12，然后按 Esc 键 | **Admin95** |
| **隐藏分区** | 磁盘实用程序 | 匹配桌面壁纸的 RGB 值 | **255025** |
| **最终网关** | 系统根目录 | 将“Goodbye.txt”中找到的字符串反转 | **REPLACEME** |

高级谜题涉及使用“录音机”以 0.5 倍速播放音频文件。通常，会有一个声音低语一串数字。这些基于音频的谜题是访问硬盘“秘密文件”区域所必需的。

<Callout type="info" title="调查笔记">
如果你在尝试打开“叔叔的保险库”之前删除了任何文件，其密码会发生改变。如果代码 8421 不起作用，请检查回收站的“属性”，查看你已删除了多少字节的数据；该数字即为新密码。
</Callout>

</Section>

<Section icon={Trophy} title="结局指南与通关清单">

Desktop Explorer 包含三个不同的结局，具体取决于你发现了多少秘密以及你如何对待“叔叔”的遗产。游戏会根据你是删除了他的个人记忆还是保留它们来记录你的“道德值”（Ethics Score）。

| 结局名称 | 达成要求 | 故事结局 |
| :--- | :--- | :--- |
| **抹除** | 关机前删除“我的文档”中的所有文件 | 电脑被清空，谜团仍未解开。 |
| **继承** | 打开“Private”文件夹但不上报调查结果 | 你保留了这台电脑，但“叔叔”开始在 UI 中作祟。 |
| **真相** | 收集全部 12 个“证据”日志并将其上传到网络 | 谜团被解开，数字幽灵得以安息。 |

要达成“真结局”，你必须非常细心。在最终系统关机之前，请使用以下清单确保你已满足所有必要条件。

<Section icon={Swords} title="技术规格与系统要求">

Desktop Explorer 旨在现代硬件上运行，同时完美模拟了 1995 年电脑的各种限制。虽然配置要求很低，但仍建议在稳定的环境下运行，以防止真实的系统崩溃干扰游戏的“虚假”蓝屏。

| 类别 | 最低配置 | 推荐配置 |
| :--- | :--- | :--- |
| **操作系统** | Windows 10 (64位) | Windows 11 (64位) |
| **处理器** | 双核 2.0 GHz | 四核 3.0 GHz |
| **内存** | 4 GB 内存 | 8 GB 内存 |
| **显卡** | 集成显卡 | 独立显卡 (1GB 显存) |
| **存储空间** | 500 MB 可用空间 | 1 GB（用于存放隐藏日志文件） |

在选项菜单中调整“CRT 滤镜”可能会显著影响性能。如果你在“磁盘碎片整理程序”运行期间遇到卡顿，请考虑降低扫描线强度。

<Callout type="info" title="显示提示">
为了获得最沉浸式的体验，请在黑暗的房间中进行游戏，并将“VHS 滤镜”设置为 75%。这会增强恐怖氛围，使虚拟操作系统发出的蓝色屏幕微光显得更加真实。
</Callout>
