<Section icon={Map} title="Explorando la interfaz del SO Retro">

Navegar por el sistema operativo simulado Windows 95 en Desktop Explorer requiere buen ojo para el diseño digital retro. La interfaz imita una nostálgica computadora personal de finales de los 90, completa con un efecto de monitor CRT parpadeante, arte de píxeles de baja resolución y sonidos de alerta del sistema clásicos. Para progresar, debes interactuar con varias aplicaciones de escritorio, leer archivos de texto, inspeccionar propiedades de imagen y buscar pistas dejadas en las carpetas del sistema.

Comprender la utilidad de cada programa virtual es fundamental para tu investigación. Algunas aplicaciones son accesibles de inmediato, mientras que otras requieren privilegios de administrador o claves de descifrado para desbloquearse.

| Programa | Propósito | Valor de investigación |
| :--- | :--- | :--- |
| Explorador de archivos | Navegar por directorios, abrir documentos | Alto (Contiene pistas principales de la historia) |
| Retro Chat | Leer mensajes archivados | Medio (Proporciona trasfondo del personaje) |
| Herramienta de recuperación | Restaurar archivos de sistema eliminados | Alto (Desbloquea áreas bloqueadas de la demo) |
| Símbolo del sistema | Ejecutar comandos del sistema | Alto (Accede a rutas de directorios ocultos) |

A medida que hagas clic en las carpetas, presta mucha atención a los tamaños de los archivos y las fechas de modificación. Las discrepancias en los metadatos del sistema a menudo apuntan a archivos o directorios ocultos que han sido alterados intencionalmente para encubrir secretos.

<Callout type="warning" title="Inestabilidad del sistema">
Prepárate para fallos repentinos del sistema. Abrir archivos corruptos puede desencadenar anomalías visuales, audio distorsionado y ventanas emergentes inesperadas que revelan pistas crípticas o mensajes de advertencia de una entidad desconocida.
</Callout>

</Section>

<Section icon={Target} title="Recorrido paso a paso: Soluciones de acertijos">

Completar la demo requiere resolver una secuencia de acertijos digitales repartidos por el sistema operativo simulado. Sigue este recorrido estructurado para eludir los protocolos de seguridad, descifrar archivos bloqueados y descubrir la verdad tras la misteriosa desaparición de tu tío.

<Steps>
  <Step num="1" title="Arranque e inspección inicial">
    Enciende la computadora virtual y espera a que se cargue el escritorio. Abre el Explorador de archivos y navega hasta la carpeta 'Mis documentos'. Abre el archivo de texto llamado 'ReadMe.txt' para encontrar tu primera pista narrativa. Luego, abre la 'Papelera de reciclaje' y anota los nombres de los archivos corruptos; necesitarás restaurarlos más adelante una vez que adquieras la herramienta de recuperación.
  </Step>
  <Step num="2" title="Descifrando el primer código">
    Localiza la carpeta bloqueada llamada 'Uncle_Private'. Para descifrarla, abre la aplicación 'Retro Chat' y revisa el historial de chat entre tu tío y un usuario llamado 'Archivist_95'. El registro del chat menciona un año de aniversario significativo. Combina este año con la clave de arranque predeterminada del sistema para formar la contraseña de desbloqueo.
  </Step>
  <Step num="3" title="Restaurando los datos eliminados">
    Inicia la 'Herramienta de recuperación' desde el Menú de inicio. Introduce la clave de registro restaurada para arrancar el asistente de recuperación. Selecciona los archivos eliminados de la lista de escaneo e inicia el proceso de extracción. Esta acción restaurará 'Uncle_Notes.txt' y un archivo de audio distorsionado en tu escritorio.
  </Step>
  <Step num="4" title="Accediendo al directorio final">
    Abre el 'Símbolo del sistema' y escribe el comando de cambio de directorio del sistema para acceder a la partición oculta del sistema. Ejecuta el archivo de audio restaurado a través de la línea de comandos para activar la secuencia final de la demo, que descifra el archivo de misterio definitivo.
  </Step>
</Steps>

Siguiendo esta secuencia, navegarás con éxito por los acertijos principales de la demo. A continuación se presenta un resumen de las contraseñas críticas y dónde encontrarlas durante tu partida.

| Archivo / Carpeta bloqueada | Ubicación de la pista | Contraseña de descifrado |
| :--- | :--- | :--- |
| Uncle_Private | Registro de chat eliminado #4 | 1995_OS |
| System_Secure_Bin | Archivo de registro 0x09 | RECOVERY_KEY |
| Archive_July17 | Nota adhesiva del escritorio | RETRO_CRT |

<Callout type="tip" title="Comando oculto">
Si te quedas atascado, abre el Símbolo del sistema y escribe 'HELP'. Esto muestra una lista de comandos del sistema ocultos que pueden eludir errores menores de archivos o revelar la ruta del directorio de documentos elusivos.
</Callout>

</Section>

<Section icon={Shield} title="Requisitos del sistema y ajustes de rendimiento">

Para apreciar plenamente la atmósfera retro de Desktop Explorer, se recomienda encarecidamente asegurarse de que la configuración de tu computadora física esté optimizada para el filtro CRT y el audio espacial del juego. El juego está diseñado para ejecutarse sin problemas en la mayoría del hardware moderno, pero ajustes específicos pueden mejorar tu inmersión.

| Especificación | Requisito mínimo | Configuración recomendada |
| :--- | :--- | :--- |
| SO | Windows 10 (64 bits) | Windows 10 / 11 (64 bits) |
| Procesador | Dual Core 2.0 GHz | Quad Core 3.0 GHz |
| Memoria | 4 GB de RAM | 8 GB de RAM |
| Gráficos | Compatible con DirectX 11 | GTX 1050 o equivalente |
| Almacenamiento | 1 GB de espacio disponible | 2 GB de espacio disponible |

Ajustar los filtros visuales en el menú principal puede afectar significativamente tu experiencia de juego. Si la distorsión de la pantalla CRT retro o el fuerte ruido de VHS causan fatiga visual, estas características pueden desactivarse o reducirse en el menú de opciones sin perder la estética central del entorno de terror digital.
