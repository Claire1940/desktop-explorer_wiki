<Section icon={Target} title="Comprendiendo la Interfaz del SO Retro">

Navegar por el paisaje digital de este juego de terror psicológico retro requiere una mirada aguda y pensamiento analítico. El núcleo de la **jugabilidad de Desktop Explorer** se centra en un monitor CRT simulado que ejecuta un sistema operativo antiguo de los años 90. Para progresar en el misterio, debes tratar el escritorio virtual como una computadora real, buscando en directorios anidados, ejecutando herramientas del sistema y analizando archivos de usuario fragmentados dejados por un pariente desaparecido.

A diferencia de los sistemas modernos, esta interfaz retro depende de la gestión manual de archivos. No puedes simplemente buscar un archivo; debes localizar la ruta exacta del directorio, restaurar sectores corruptos y desencriptar archivos bloqueados usando pistas escondidas a plena vista.

<Callout type="tip" title="Consejo de Navegación del Sistema">
Revisa siempre las extensiones de los archivos. Algunos archivos pueden parecer configuraciones del sistema, pero en realidad son archivos de texto renombrados que contienen contraseñas críticas o detalles de la historia.
</Callout>

Para ayudarte a navegar por el sistema de manera eficiente, familiarízate con las aplicaciones principales disponibles en el escritorio:

| Aplicación | Función | Prioridad |
| :--- | :--- | :--- |
| Explorador de Archivos | Navegar por directorios, abrir archivos de texto y buscar carpetas ocultas | Alta |
| Cliente de Chat | Leer conversaciones archivadas para reconstruir las relaciones entre personajes | Alta |
| Papelera de Reciclaje | Recuperar documentos eliminados, registros del sistema y archivos multimedia corruptos | Media |
| Reproductor Multimedia | Reproducir clips de audio retro y archivos de video con pistas del entorno | Media |
| Herramienta de Desencriptación | Introducir claves hexadecimales y contraseñas para desbloquear directorios seguros | Alta |

Entender cómo interactúan estos programas es la base del ciclo de juego. Por ejemplo, una pista encontrada en un registro de chat podría proporcionar la contraseña necesaria para la Herramienta de Desencriptación, que posteriormente desbloquea un archivo de video corrupto en el Reproductor Multimedia.

</Section>

<Section icon={Map} title="Jugabilidad de Desktop Explorer: Guía Paso a Paso">

Para completar con éxito la investigación principal, debes seguir un camino estructurado de forense digital. Esta guía cubre los pasos esenciales de progresión para guiarte desde la secuencia de inicio inicial hasta el corazón del misterio digital.

<Callout type="warning" title="Pistas que se pueden perder">
Eliminar ciertos archivos del sistema antes de desencriptarlos puede bloquearte permanentemente de finales específicos. Siempre haz una copia de seguridad de los archivos sospechosos en la unidad de disquete virtual.
</Callout>

Sigue estos pasos para progresar a través de la investigación al principio y a mitad del juego:

<Steps>
  <Step num="1" title="Inicia el Sistema y Mapea los Directorios">
    Enciende la computadora virtual y espera a que cargue el escritorio. Abre el Explorador de Archivos y abre sistemáticamente cada carpeta en el directorio raíz C:. Toma notas físicas de cualquier carpeta o archivo bloqueado que requiera privilegios de administrador.
  </Step>
  <Step num="2" title="Analiza los Archivos de Chat">
    Abre el Cliente de Chat y lee los registros entre el usuario y su tío. Busca fechas resaltadas, ubicaciones y frases inusuales. El año de nacimiento del tío, encontrado en una conversación de chat de junio de 1998, sirve como tu primera contraseña importante.
  </Step>
  <Step num="3" title="Restaura la Papelera de Reciclaje">
    Navega a la Papelera de Reciclaje y localiza el archivo llamado `sys_recovery.log`. Restaura este archivo al escritorio. Ábrelo usando el editor de texto para revelar una lista de directorios eliminados y sus ubicaciones de sector originales.
  </Step>
  <Step num="4" title="Ejecuta el Programa de Desencriptación">
    Abre la Herramienta de Desencriptación en el escritorio. Introduce la contraseña obtenida de los archivos de chat para desbloquear la carpeta segura del sistema. Esta acción te otorga acceso de administrador a la unidad D: oculta.
  </Step>
</Steps>

Una vez que hayas desbloqueado los privilegios de administrador, se presentarán varios desafíos de acertijos. Usa las siguientes soluciones de contraseñas para omitir estos bloqueos de seguridad:

| Ubicación de Contraseña | Fuente de la Pista | Código de Solución | Desbloquea |
| :--- | :--- | :--- | :--- |
| Carpeta Bloqueada 1 | Fecha del registro de chat | 1958 | Archivos del diario personal del tío |
| Registro de Chat Encriptado | Cifrado Vigenere | RETRO95 | Archivo de la conversación final |
| Configuración del Sistema | Código hexadecimal decimal | 4F8E | Archivos de registro corruptos |
| Archivo Zip | Metadatos de archivo de imagen | BLUE_SCREEN | Grabación de audio oculta |

</Section>
