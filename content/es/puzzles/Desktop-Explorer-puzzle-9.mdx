<Section icon={Target} title="Guía Paso a Paso para el Acertijo 9 de Desktop Explorer">

Desbloquear el directorio del sistema cifrado requiere ejecutar una secuencia precisa de comandos y manipulaciones de archivos. Sigue esta guía paso a paso para resolver el **acertijo 9 de Desktop Explorer** sin activar un bloqueo de seguridad del sistema.

<Steps>
  <Step num="1" title="Activar el Evento de Generación">
    Abre tu cliente de correo electrónico virtual y lee el mensaje del "Remitente Desconocido". Esta acción activa la generación del directorio oculto en tu unidad C virtual.
  </Step>
  <Step num="2" title="Localizar la Carpeta Oculta">
    Abre la aplicación Terminal de Comandos desde tu escritorio. Escribe `cd C:\` y presiona enter, luego ejecuta el comando `dir /a` para mostrar todos los directorios ocultos del sistema. Verás aparecer una carpeta llamada `_SYS09`.
  </Step>
  <Step num="3" title="Recuperar el Registro Corrupto">
    Minimiza la terminal y abre la Papelera de Reciclaje en tu escritorio. Localiza el archivo llamado `TRASH_09.TMP`, haz clic derecho sobre él y selecciona "Restaurar" para enviarlo de vuelta a su carpeta original del sistema.
  </Step>
  <Step num="4" title="Calcular la Clave Hexadecimal">
    Abre el archivo `TRASH_09.TMP` restaurado usando la aplicación Bloc de Notas. Encontrarás una cadena hexadecimal de 8 dígitos. Usa la tabla de conversión que se encuentra en el archivo de configuración del sistema para calcular tu clave de descifrado única.
  </Step>
  <Step num="5" title="Sobrescribir el Registro del Sistema">
    Inicia `REGEDIT.EXE` desde la carpeta del sistema. Navega hasta `HKEY_LOCAL_MACHINE\SYSTEM09`, haz doble clic en el valor "DecryptionKey" e ingresa tu código calculado para desbloquear permanentemente el directorio.
  </Step>
</Steps>

| Paso | Acción | Resultado Esperado |
|------|--------|-----------------|
| 1 | Leer Correo | Activa la generación de archivos en segundo plano |
| 2 | Ejecutar Comando de Terminal | Revela el directorio oculto `_SYS09` |
| 3 | Restaurar Archivo TMP | Coloca la referencia de descifrado en las carpetas del sistema |
| 4 | Ejecutar Lógica de Descifrado | Genera la clave de registro única de 8 dígitos |
| 5 | Modificar Registro | Otorga acceso completo a la carpeta bloqueada |

<Callout type="warning" title="Advertencia del Registro">
No modifiques ningún valor del registro que no sea "DecryptionKey". Alterar valores aleatorios en `REGEDIT.EXE` puede provocar un fallo simulado del sistema, obligándote a reiniciar el sistema operativo virtual y perder el progreso no guardado del acertijo.
</Callout>

Una vez que el registro haya sido modificado, puedes navegar de regreso a tu explorador de archivos de forma segura. La carpeta antes bloqueada ahora aparecerá como un directorio estándar y accesible que contiene documentos críticos de la trama.

</Section>

<Section icon={Shield} title="Pistas Críticas y Ubicaciones de Archivos">

Para completar con éxito el proceso de descifrado, debes reunir varias pistas digitales ocultas dentro de la computadora simulada. Estos archivos contienen las variables necesarias para resolver la parte matemática del acertijo.

<CardGrid cols={3}>
  <Card title="Configuración SYSTEM.INI">
    - **Ubicación**: `C:\WINDOWS\SYSTEM.INI`
    - Contiene el valor de desplazamiento base necesario para la conversión hexadecimal.
    - Crucial para calcular la clave final.
  </Card>
  <Card title="CHAT_LOG_09.TXT">
    - **Ubicación**: `C:\DOCUMENTS\CHAT_LOG_09.TXT`
    - Revela la conversación con pistas de seguridad entre tu tío y un usuario misterioso.
    - Explica el orden de la secuencia.
  </Card>
  <Card title="Recuperación de la PAPELERA_DE_RECICLAJE">
    - **Ubicación**: Papelera de Reciclaje Virtual
    - Contiene el archivo corrupto `TRASH_09.TMP`.
    - Debe restaurarse antes de analizar la cadena hexadecimal.
  </Card>
</CardGrid>

La conversión hexadecimal depende en gran medida de la fecha de instalación del sistema operativo virtual, que se puede encontrar en el panel de propiedades del sistema. Usa la tabla a continuación para mapear las variables a tu instancia específica del juego.

| Variable | Archivo de Origen | Propósito |
|----------|-------------------|-----------|
| Cadena Hex Base | `TRASH_09.TMP` | El código bruto que necesita descifrado |
| Valor de Desplazamiento | `SYSTEM.INI` | El número restado del código hex bruto |
| Fecha de Instalación | Propiedades del Sistema | Determina el factor de multiplicación final |
| Pista de Seguridad | `CHAT_LOG_09.TXT` | Confirma el orden correcto de los dígitos |

<Callout type="tip" title="Consejo de Descifrado">
Si tu cadena hex base comienza con "A", tu valor de desplazamiento siempre se duplica. Verifica siempre el primer carácter de tu código bruto antes de realizar cualquier cálculo.
</Callout>

Al organizar estas pistas, eliminas la necesidad de adivinar. El acertijo se basa en pura lógica, reflejando la filosofía de diseño de los juegos de aventuras gráficas clásicos de los 90.

</Section>
