<Section icon={Trophy} title="Guía de Progresión Paso a Paso">

Navegar por las capas iniciales del juego Desktop Explorer requiere un enfoque estructurado. Los siguientes pasos describen el procedimiento estándar para romper el primer nivel de seguridad y acceder a la red de carpetas secundaria.

<Steps>
  <Step num="1" title="Inicialización del Sistema">
    Inicia la terminal virtual e inspecciona los archivos predeterminados en el escritorio. Abre primero el archivo Readme para entender los parámetros iniciales y las pautas establecidas por el administrador del sistema.
  </Step>
  <Step num="2" title="Búsqueda en el Directorio">
    Navega a la carpeta de documentos. Abre cada subdirectorio y localiza el archivo de archivo encriptado que requiere una contraseña para continuar.
  </Step>
  <Step num="3" title="Correlación de Pistas">
    Cruza las pistas encontradas en el registro del sistema con las fechas de creación de los archivos. Las diferencias en las marcas de tiempo proporcionan la secuencia numérica requerida para el descifrado.
  </Step>
  <Step num="4" title="Descifrado de Carpetas">
    Introduce la contraseña calculada en el aviso del archivo. Extrae el contenido para revelar la siguiente capa del directorio, que contiene herramientas avanzadas de diagnóstico del sistema.
  </Step>
</Steps>
