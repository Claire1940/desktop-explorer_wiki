<Section icon={Terminal} title="Step-by-Step Decryption Walkthrough">

Follow these precise steps to bypass the security on Archive_08.dat. This sequence assumes you have already restored the Command Prompt and gathered the necessary files mentioned in the previous section.

<Steps>
  <Step num="1" title="Initialize the Terminal">
    Open the Command Prompt from your desktop. Type `cd C:\WINDOWS\SYSTEM\TEMP` and press Enter. You should see the directory change. If the directory is not found, you haven't enabled "Show Hidden Files" in the View menu of the File Explorer.
  </Step>
  <Step num="2" title="Identify the Offset">
    Open "Mail_Log.txt" in the Notepad application. Locate the last entry from "Uncle." Note the timestamp (usually 11:42 PM). The "offset" for the puzzle is the numerical difference between the hours and minutes (42 - 11 = 31).
  </Step>
  <Step num="3" title="Extract the Binary String">
    Open "System.ini" and find the line starting with `A8_KEY=`. You will see a string of 8 digits. Apply your offset (31) to the first two digits of this string by adding them together.
  </Step>
  <Step num="4" title="Execute Decryption Command">
    Return to the Command Prompt. Type the command `DECRYPT -f Archive_08.dat -k [YourCalculatedKey]`. Replace the bracketed text with your final 8-digit sequence.
  </Step>
  <Step num="5" title="Access the Contents">
    Once the "Decryption Successful" message appears, a new folder named "RECOVERED_DATA" will appear on your desktop. Open it to complete Desktop Explorer puzzle 8.
  </Step>
</Steps>

| :--- | :--- | :--- |
| **DIR /H** | Shows hidden files in current folder | List including Archive_08.dat |
| **HELP DECRYPT** | Displays command parameters | Usage: DECRYPT -f [file] -k [key] |
| **CLS** | Clears the terminal screen | Empty terminal window |
