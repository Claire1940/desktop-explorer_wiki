<Section icon={Target} title="Step-by-Step Hourglass Solution Walkthrough">

Solving the **Desktop Explorer hourglass puzzle** requires a specific sequence of actions within the virtual BIOS and Desktop environments. Follow these steps to break the loading loop and access the encrypted data.

<Steps>
  <Step num="1" title="Locate the Corrupted Log">
    Navigate to the `C:\TEMP` directory and find a file named `BOOT_ERR.LOG`. Open it using the Text Editor. Scroll to the bottom to find a timestamp marked "CRITICAL FAILURE." Note the exact date and time (usually July 17, 1995, at 03:14 AM).
  </Step>
  <Step num="2" title="Access System Settings">
    Click on the "Start" menu and select "Settings," then "Control Panel." Open the "Date/Time" utility. This is where the primary manipulation for the **Desktop Explorer hourglass puzzle** occurs.
  </Step>
  <Step num="3" title="Synchronize the Clock">
    Manually change the system date to match the timestamp found in Step 1. You must be precise within a one-minute margin. Once the values are entered, do not click "Apply" yet.
  </Step>
  <Step num="4" title="Execute Time_Sync.exe">
    While the Date/Time window is open, run `TIME_SYNC.EXE` from the `C:\DOS` folder. The hourglass cursor should begin to flicker between the standard pointer and the loading icon.
  </Step>
  <Step num="5" title="Confirm the Breach">
    Click "Apply" on the Date/Time window. The screen will momentarily glitch, and the "Classified" folder will finally open, revealing the hidden project files.
  </Step>
</Steps>

<Section icon={Shield} title="Troubleshooting Common Puzzle Errors">

Many players encounter issues where the hourglass persists even after changing the clock. This usually happens because of background processes that "lock" the system time.

**Common Issues and Fixes:**
- **The Clock Resets**: This happens if you haven't deleted the `AUTOEXEC.BAT` file in the root directory. Delete it to allow permanent time changes.
- **Hourglass is Frozen**: If the animation stops entirely, the game has soft-locked. You must use the `CTRL+ALT+DEL` command within the game to restart the virtual shell.
- **Missing Date/Time Icon**: If the icon is missing from the Control Panel, you must find the `TIMEDATE.CPL` file in the Recycle Bin and "Restore" it.

<Callout type="success" title="Pro Strategy">
