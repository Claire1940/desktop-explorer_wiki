<Section icon={Target} title="Step-by-Step Walkthrough for Puzzle 5">

Follow these precise steps to resolve the **Desktop Explorer puzzle 5** and unlock the next chapter of the mystery. Accuracy is paramount, as entering the wrong password three times will lock the system for two real-time minutes.

<Steps>
  <Step num="1" title="Initialize the Recovery Tool">
    Navigate to the 'C:/Programs/System' folder and launch the 'Recovery.exe' application. You will be prompted for a 'Maintenance Code.' Enter the date found on the 'System Property' screen (07-17-1996).
  </Step>
  <Step num="2" title="Reconstruct the Corrupted Image">
    Open the 'Gallery' and look for 'IMG_005.bmp.' It will appear as static. Use the Hex Editor to change the first four lines of code to match the standard BMP header format. This reveals a photo of a desk with a sticky note.
  </Step>
  <Step num="3" title="Extract the Sticky Note Clue">
    Zoom into the reconstructed image. The sticky note contains a string: 'B-Day + 104'. Cross-reference this with the birthday found in the 'Personal' folder to get a 4-digit code.
  </Step>
  <Step num="4" title="Bypass the Firewall">
    Run the 'Command Prompt' and type 'ping secret.server.net'. This will return an IP address. The last three digits of this IP are the final offset needed for the decryption tool.
  </Step>
  <Step num="5" title="The Final Decryption">
    Combine the sticky note code and the IP offset. Enter this into the 'Encrypted Archive' popup. If successful, the folder will unlock, revealing the 'Truth.pdf' file.
  </Step>
</Steps>
