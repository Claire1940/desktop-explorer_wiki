<Section icon={Target} title="Decrypting Passwords and Solving Basic Puzzles">

Progression in the **Desktop Explorer indie game** relies entirely on your ability to locate and decrypt passwords. These passwords are not randomly generated; they are hidden within the metadata of images, the text of deleted emails, or obfuscated inside system logs. 

To systematically crack these security measures, players must learn to read the virtual environment carefully. The table below details the primary directories you must access, where to find their clues, and the general decryption method required.

| Directory | Clue Source | Decryption Method | Difficulty |
| :--- | :--- | :--- | :--- |
| **Root/Documents** | Readme.txt hints | Simple cipher | ★★☆☆☆ |
| **System32/Drivers** | Hex dump of logs | Binary translation | ★★★☆☆ |
| **User/Downloads** | Corrupt image files | Metadata inspection | ★★★★☆ |
| **Network/Shared** | Packet capture log | Port mapping | ★★★★★ |

To unlock your very first password-protected folder and begin your journey, follow this step-by-step decryption sequence.

<Steps>
  <Step num="1" title="Locate the Readme File">
    Navigate to the `C:/User/Documents` folder. Open the `Readme.txt` file. Note the capitalized letters scattered anomalously throughout the text.
  </Step>
  <Step num="2" title="Extract the Cipher Key">
    Write down the capitalized letters in order. This sequence forms a simple Caesar cipher shift key (typically a shift of +3 or -3 depending on your game seed).
  </Step>
  <Step num="3" title="Inspect Image Metadata">
    Right-click the image file named `hint.bmp` in the same directory and select 'Properties'. Under the metadata tab, locate the encrypted string in the comment section.
  </Step>
  <Step num="4" title="Apply the Shift and Unlock">
    Apply your Caesar cipher key to the metadata string. Input the resulting plaintext password into the locked folder prompt to grant access to the next directory tier.
  </Step>
</Steps>

<Callout type="tip" title="Hidden File Extensions">
Always enable "Show Hidden Files" in the system settings panel immediately upon starting the game. Several key puzzle clues are located in hidden `.ini` and `.sys` files that do not render under default view settings.
</Callout>
