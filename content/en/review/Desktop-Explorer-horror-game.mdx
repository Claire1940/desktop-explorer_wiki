<Section icon={Map} title="Getting Started with Desktop Explorer">

The **Desktop Explorer horror game** is a unique simulation-based adventure that transports players back to the mid-90s. You find yourself sitting in front of a flickering CRT monitor, exploring a discarded computer that holds the secrets of a missing person. Unlike traditional horror games that rely on jump scares in a 3D environment, this title builds dread through file exploration, corrupted media, and the realization that you are not alone in the digital space.

To succeed, you must treat the simulated OS as a real investigation tool. Every folder, no matter how mundane it looks, could contain a crucial piece of the puzzle. The game rewards curiosity but punishes carelessness, as some system files are trapped or monitored by unknown entities.

| Interface Element | Primary Function | Investigation Value |

| **Media Player** | Plays discovered audio and video files | Medium: Visual clues for puzzles |
| **Control Panel** | Adjusts system settings and security | Low: Used for specific late-game puzzles |

The investigation generally follows a non-linear path, but there are specific milestones you must reach to trigger the next phase of the mystery. Below is the recommended sequence for your first playthrough.

<Steps>
  <Step num="1" title="Initial Desktop Exploration">
    Start by opening every file on the desktop. Read the "ReadMe.txt" file and check the "Work" folder. You will find a locked PDF that requires a 4-digit code found in the "Notes" application.
  </Step>
  <Step num="2" title="The Deleted Files Phase">
    Access the Recycle Bin. You will find several corrupted .jpg files. Use the built-in "Repair Tool" found in the System folder to view these images, which reveal a hidden location in the real world.
  </Step>
  <Step num="3" title="Chat Log Analysis">
    Open the "Instant Messenger" archive. Search for keywords like "password," "secret," or "Uncle." These logs reveal the strained relationship between the computer's owner and their family.
  </Step>
  <Step num="4" title="Deep System Access">
    Once you find the Admin password (usually hidden in the 'Config.sys' file viewable via Notepad), you can access the 'Private' partition. This is where the horror elements significantly intensify.
  </Step>
</Steps>

<Callout type="tip" title="Expert Tip">

<Section icon={Trophy} title="Endings and Requirements Guide">

There are three primary endings in the **Desktop Explorer horror game**. Your final outcome is determined by how much of the "Forbidden Data" you managed to recover and whether you chose to delete the system or upload its contents to the "Cloud" at the end of the game.

| Ending Name | Requirements | Narrative Outcome |
| :--- | :--- | :--- |
| **The Ignorance Ending** | Complete the game with < 50% files found | You escape the computer, but the mystery remains unsolved |
| **The Digital Ghost Ending** | Upload the 'Private' folder to the web | The entity is released, and your character is trapped |
| **The True Resolution** | Find all 12 'Uncle' files and delete the OS | The cycle is broken, and the missing person is found |

<Callout type="success" title="Achievement Unlocked">
