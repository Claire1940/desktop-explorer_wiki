<Section icon={Target} title="Advanced File Manipulation Techniques">

The core of the **Desktop Explorer game guide** involves manipulating the game's files directly on the desktop. This is most evident during the "Big Door" puzzle, where a standard key will not work. You must use the "image is worth a thousand words" clue to bypass the lock.

<Steps>
  <Step num="1" title="Locate the Key File">
    Find the file named `key.stp` or similar in your downloads or puzzle folder.
  </Step>
  <Step num="2" title="Rename the Extension">
    Right-click the file and rename it to `image.png` or `image.jpg`. This "tricks" the OS into revealing the hidden visual data.
  </Step>
  <Step num="3" title="Extract the Password">
    Open the newly created image. The text "Every lock obeys portonus" will appear. Use **portonus** to unlock the central door.
  </Step>
  <Step num="4" title="Check Permissions">
    If the system denies the rename, ensure you are logged in as the correct user or check if the file is "read-only" in the properties.
  </Step>
</Steps>

<Callout type="info" title="System Mechanic">
Renaming files creates a new copy in this OS. If you make a mistake, you can always delete the corrupted copy and try again with the original source file.
</Callout>

</Section>
