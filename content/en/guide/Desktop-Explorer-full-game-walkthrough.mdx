  <Step num="2" title="The Password Challenge">
    Access the "Mail Client" application. You will find a series of unsent drafts. The subject lines of these drafts, when read in chronological order, provide the hint for the BIOS password. Entering this password allows you to view "System32" hidden files.
  </Step>
  <Step num="3" title="Recovering Deleted Evidence">

  </Step>
  <Step num="4" title="The Chat Log Investigation">

  </Step>
  <Step num="5" title="Final System Override">

  </Step>

Advanced puzzles involve using the "Sound Recorder" to play back audio files at 0.5x speed. Often, a voice will whisper a sequence of numbers. These audio-based puzzles are mandatory for accessing the "Secret Files" section of the hard drive.

<Callout type="info" title="Investigation Note">
The "Uncle's Vault" password changes if you delete any files before attempting to open it. If the code 8421 does not work, check the "Properties" of the Recycle Bin to see how many bytes of data you have removed; that number is the new code.
